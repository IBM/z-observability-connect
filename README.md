# Z Observability Connect

Welcome to the Z Observability Connect public asset page. Each available asset is organized into its own folder. These assets are supplemental to officially supported code and documentation and are intended to provide guidance and examples.

## Available Assets:

### [Proof of Concept Guide](https://github.com/IBM/z-observability-connect/tree/main/proof-of-concept-guide/poc-guide.md)
A step-by-step, cookbook-style guide for setting up a Z Observability Connect Proof of Concept (PoC) in a non-production environment.

### [Grafana Sample Dashboards](https://github.com/IBM/z-observability-connect/tree/main/grafana-dashboards)

Sample Grafana dashboards are provided to help visualize and validate telemetry generated by the Z Observability Connect (ZOC) Telemetry Controller. These dashboards serve as examples and as a starting point for exploring the z/OS metrics and log data available through ZOC.
